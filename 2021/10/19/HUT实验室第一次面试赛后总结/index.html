T1 A

有趣的题面：

jelllllllll和tzjjjjjjjj是好朋友，他们都很会打枪
这里有 N个枪靶，下标从1到N。
每进行一次，轮到的那个人至少打掉一个，最多能打掉K个下标连续的靶子
每次都是从jelllllllll开始游戏，然后tzjjjjjjjjjjjjj，接着jelllllllll….依次进行，如果轮到谁打没有靶子了，另外一个人就获得胜利
奥对了,jelllllllll的游戏名是”Adrien”,tzjjjjjjjj的游戏名是”Austin”.

Input

仅一行,两个整数 N, K (0 ≤ N ≤ 106, 1 ≤ K ≤ 106).

Output

打印获得胜利的人的游戏名（不带引号）

原题链接：Dashboard - 2018-2019 ACM-ICPC, Asia Nanjing Regional Contest - Codeforces

  • 思路：

    • 当k=0时，Adrien永远打不到最后一个靶子，先手必输
    • 当k=1时，获胜与枪靶的奇偶有关，奇数则Adrien胜，偶数则Austin胜
    • 当k!=1时，Adrien在第一步打靶时可以从中间打起，将枪靶分为左右对称两部分，这样只要Austin拿什么，Adrien就可以在另一半区间进行相同操作，保证Adrien一定能赢

T2 B

有趣的题面：

堆堆和栈栈是两个好朋友，这天他们遇到了好朋友团子。
堆堆有一些带编号的苹果，栈栈也有一些
堆堆的那堆苹果编号没有重复的，栈栈的也一样
他们决定将两人的苹果合并，并且在合并后的那堆苹果中每一个不同编号的苹果拿一个出来给团子
团子可高兴了，你能帮助她根据编号大小从小到大例举出能获得哪些编号的苹果吗

Input

多组输入！每组输入数据分为三行,第一行有两个数字n,m(0 < n,m <= 10000),分别表示堆堆苹果数量n和栈栈苹果数量m.后两行分别表示堆堆和栈栈苹果的编号.每个元素为不超出int范围的整数,每个元素之间有一个空格隔开.

Output

针对每组数据输出一行数据,表示合并后的列举出的编号,要求从小到大输出,每个元素之间有一个空格隔开，没有多余的空格！

注意：多组输入，The input is terminated by the end of file （EOF)

c语言 ：while(scanf(“%d%d”,&n,&m)!=EOF)
c++ : while(cin >> n >> m)

原题链接：Problem - 1412 (hdu.edu.cn)

  • 思路：

    将两个区间合并后从小到大输出所有不同的数

T3 C

有趣的题面：

n个英雄，m个怪物，每个英雄都有一个能杀死的怪物的集合，每个英雄只能杀死一只怪物。
现在有一个强大的魔法师刘放提供了k瓶神奇药水，使得携带药水的英雄能够多杀死一只怪物，但是每个英雄只能携带一瓶，问你最多能够杀死多少只怪物。

Input

第一行输入n，m，k (1 ≤ n, m, k ≤ 500)
接下来n行，每行一个num，表示集合中数字个数，接下来num个数，表示怪物编号

Output

输出一行，表示最多杀死怪物的个数

原题链接：Dashboard - 2018-2019 ACM-ICPC, Asia Nanjing Regional Contest - Codeforces

  • 思路：

    是不该出现在一面的网络流呢

    每个英雄与源点连边，流量为1

    每个怪物与汇点连边，流量为1

    设置一个虚拟的点，与源点连边流量为k，与每个英雄连边流量为1

T4 D

有趣的题面：

Alice 和 Bob 喜欢玩游戏。现在，他们准备开始一个新的游戏。他们已经沿直线放置了 n 块巧克力。Alice 开始从左向右逐块吃巧克力，同时 Bob — 从右向左。对于每块巧克力，玩家吃掉它所需的时间是已知的 (Alice 和 Bob 以相等的速度吃它们)。当玩家吃完一块巧克力时，他立即开始吃另一块。不允许在同一时刻吃两块巧克力，不允许吃后剩余一部分，也不允许暂停吃巧克力。如果两个玩家同时开始吃同一块巧克力，那么 Bob 会把它留给 Alice，因为 Bob 要做一位真正的绅士。

每个玩家将吃掉多少块巧克力？

Input

第一行包含一个整数 n (1 ≤ n ≤ 105) — 桌子上的巧克力数目。第二行包含一个序列 t1, t2, …, t**n (1 ≤ t**i ≤ 1000)，其中 t**i 是吃完第 i 块巧克力 (按从左向右的顺序) 所需的时间 (按秒)。

Output

打印两个数字 a 和 b，其中 a 是 Alice 吃掉的巧克力数目，且 b 是 Bob 吃掉的巧克力数目。

原题链接：Problem - 6C - Codeforces

  • 思路：

    依次枚举每一块巧克力，将Alice正在吃的巧克力与Bob正在吃的巧克力做对比

    每次吃完相对较小的那块巧克力，再将吃了一半的巧克力与另一边的下一块巧克力进行比较

T5 E

不有趣的题面：

给一个由 n 个整数组成的数组 。
你最多可以从此数组中删除一个元素，当然也可以不删。
你需要计算剩余数组的严格递增的连续子数组的可能的最大长度。
从 l 到 r 的连续子数组 a 是 a[l…r]=al,al+1,…,ar。 如果 al < al+1 < ⋯ < ar，则子数组 a[l…r] 被称为严格递增。

Input

第一行包含一个整数 n (1 ≤ n ≤2e5) — 数组的长度。
第二行包含一个序列 a1, a2, …, an (1 ≤ ti ≤ 1e9)，其中 ai 是数组中第i个元素。

Output

打印一个整数——删除最多一个元素后严格递增的数组 a 的连续子数组的可能的最大长度。

原题链接：Problem - 1272D - Codeforces

  • 思路：

    用一个数组记录原数组到目前这一位为止的连续递增的位数

    枚举每一位，判断如果这一个数被删掉最长连续递增数会变为多少

    被删掉的数只可能为到目前为止连续递增位数为1的或这个数的后一位连续递增位数为1

T6 F

有趣的题面：

总所周知，摸鱼王中王暑假去叙利亚做兼职去了！

叙利亚社会在长期多元的族群构成中，已形成了自己独特的社会、文化与宗教模式，但随着危机爆发并深化！

但是，胆小的摸鱼王只在叙利亚战场应聘了一个统计员工作。战场中每天都会有人阵亡和加入，摸鱼王的工作就是记录人数。

这个工作太简单了，他为了让你少摸鱼，特地从叙利亚发来信息让你帮他写一个程序，这个程序的作用就是找出叙利亚战场人数最多的一天并汇报人数交给上级拿工资。

Input

第一行包含一个数字 n (2 ≤ n ≤ 1000)——摸鱼王在叙利亚兼职的天数。
然后是 n 行，每行包含两个整数 ai 和 bi (0 ≤ ai, bi ≤ 1000)——第 i 天阵亡的人数，以及第 i 天加入战场的人数。

Output

打印一个整数，表示他需要汇报给上级的人数。

原题链接：Problem - 116A - Codeforces

  • 思路：

    用一个变量sum记录当前在战场上的人数，增加每日加入人数，减少每日阵亡人数
    实时比较更新最大人数

不有趣的题面：

春天是鲜花的季节，水仙花就是其中最迷人的代表，数学上有个水仙花数，他是这样定义的：
“水仙花数”是指一个三位数，它的各位数字的立方和等于其本身，比如：153=1^3+5^3+3^3。
现在要求输出所有在m和n范围内的水仙花数。

Input

输入数据有多组，每组占一行，包括两个整数m和n（100<=m<=n<=999）。

Output

对于每个测试实例，要求输出所有在给定范围内的水仙花数，就是说，输出的水仙花数必须大于等于m,并且小于等于n，如果有多个，则要求从小到大排列在一行内输出，之间用一个空格隔开;
如果给定的范围内不存在水仙花数，则输出no;
每个测试实例的输出占一行。

原题链接：Problem - 2010 (hdu.edu.cn)
